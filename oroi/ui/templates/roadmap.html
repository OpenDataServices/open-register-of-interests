Adding more data

Adding a new scraper is a few days of software development work; if we’re able to find funding for that, then we’ll add any data from any UK body that requires members to make statutory declarations of interest.

There three main areas of development here:

  • Adding data from websites that are similar to those already scraped (this is the easiest one to do!)
  • Adding data from websites that are dissimilar, but that fundamentally are just a bunch of data in an HTML table (this is a bit harder)
  • Adding data in new file formats; some places publish Word docs with tables, spreadsheets, and PDFs. Cracking them open and getting the data out is hard, but worthwhile.
