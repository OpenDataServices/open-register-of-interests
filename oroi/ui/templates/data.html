Data

Once a month, declared. accesses a range of official websites and collects information on declared interests. We also use MySociety’s parliamentary data feed, to collect information on Westminster.

This information is presented in a wide range of formats, from handwritten, scanned PDFs to consistent websites. Our open-source software scrapes the websites, and converts the information into consistent data, using an early-stage data standard.

We then make the data available under an open license, so that anyone can use it, for any purpose.

If there’s data that you’d like to see, then there’s ways to get it added. Get involved
